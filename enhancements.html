Enhancements.

Current enhancements on website.

  1. Added tab icon
    • This makes new use of the rel attribute to use a .ico file for the tab icon, creating a more unified appearance of the website.
    • Code used: <link rel="icon" type="image/x-icon" href="images/favicon.ico">
    • No hyperlink needed, visible everywhere
  2. Added custom CSS selector to right align JSON logo figure element
    • Use of various css properties to set the image and figcaption to the right of the header, creating a cleaner and more appealing visual
    • code used: img.logo{
      width: 80px;
      float: right;
      position: relative;
      size: 20%;
      top: -95px;
      left: -15px;
      }

      figcaption.logo{
      text-align:left;
      float: right;
      position: relative;
      top: 0px;
      left: +100px;
      }
    • Visible on all pages, header introduced on home page
