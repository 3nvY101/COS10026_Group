Current Website Enhancements.

  1. Added tab icon

    • This makes new use of the rel attribute to use a .ico file for the tab icon, creating a more unified appearance of the website.
    • Code used:

          <link rel="icon" type="image/x-icon" href="images/favicon.ico">

    • No hyperlink needed, visible everywhere

  3. File conpression using Minification

Dramatically improving the site's speed and accessibility, directly translating into a better user experience. Essentailly converting this large easy to read code to a compress form that allows computers to read faster.
